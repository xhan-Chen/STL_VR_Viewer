Detailed Description

Primary application window for loading, editing, rendering, and viewing STL assemblies in VR.

The class owns the desktop VTK scene and the Qt controls around it. It loads STL files into a ModelPartList tree, applies per-part materials and filters, manages grouping and transforms, and starts/stops VRRenderThread for the headset view.
