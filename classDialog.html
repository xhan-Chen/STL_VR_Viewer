Detailed Description

Modal Qt dialog for editing a part's display options.

Dialog owns the widgets created from dialog.ui and adds a small set of custom controls for glow highlighting. Getter and setter methods are used by MainWindow to initialise the dialog from the selected part and read the final accepted state.
